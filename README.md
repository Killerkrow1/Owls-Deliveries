<center><h3>A mod that introduces a Owl-based delivery system to Minecraft!</h3></center>

## Features

- Adds Owls to Minecraft!
- They can be found in forests, taigas or snowy taigas, only at Night.
- There a few variants, some of which are found only in certain biomes. Most varaints also have different Eye Colors. There's also a rare Shiny Owl variant that has 1/200 chance of replacing any naturally spawned owl.
- You can tame Owls by feeding them Raw Rabbit. Breed them with Rabbit Foot.
- Rename a bundle with the name of an online player, right-click an Owl with the bundle, and it will deliver the bundle to that player.
- Renaming an Owl to "Folly", "Lux" or "Nycto", it'll get a special variant. After an Owl gets a special variant you can rename it to whatever you want.
